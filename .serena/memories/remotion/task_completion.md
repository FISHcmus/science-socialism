# Task Completion Checklist

When a coding task is completed:

1. Verify Remotion Studio still starts: `bun run studio`
2. Check for TypeScript errors (Remotion's webpack will catch these on build)
3. No separate linter/formatter configured yet
4. No test suite — this is a video project, verify visually in Remotion Studio
5. Web preview should also work: `bun run web`
