# Suggested Commands

All commands run from `semester2/science-socialism/de_an_quay_video/remotion/`.

## Development
- `bun run studio` — Launch Remotion Studio (browser-based editor/preview, port 3456)
- `bun run web` — Launch web-only preview with @remotion/player (port 3457)

## Rendering
- `bun run render` — Render full quality MP4 to `out/video.mp4`
- `bun run render:draft` — Render half-res draft to `out/draft.mp4` (faster)

## Package Management
- `bun add <pkg>` — Add dependency
- `bun install` — Install all deps

## System Utils
- `git` — version control
- `locate` — find files (preferred over `find`)
- `pdftotext` — extract text from PDFs
- `tailscale ip -4` — get Tailscale IP (100.64.0.2)
